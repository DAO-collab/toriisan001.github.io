rpmコマンドで色々確認してみる

コマンド

init 6

ランレベル「6」はシステムの再起動を意味します。

rpm

rpm -e [RPMパッケージ名]

rpmコマンドでパッケージをアンインストールするときは「-e」オプションを使用します。

rpm -qa

インストール済みのすべてのrpmパッケージの一覧を表示するときは「-qa」オプションを使用します。

「rpm -qa | grep kernel | sort」でインストールされている全パッケージの中から特定の名前を含むパッケージを検索出来る。

rpm -q [rpmパッケージ名]

例:rpm -q rpm

パッケージ名を表示

インストールされていないパッケージ名を指定すると「yum-utils([rpmパッケージ名])パッケージがインストールされていません」

rpm -ql [rpmパッケージ名]

例:rpm -ql rpm

パッケージに含まれるファイルの一覧を表示

「rpm -ql rpm | grep bin」のようにしてgrepしてもよい

rpm -qf ${ファイル名}

例:rpm -qf /etc/httpd/conf/httpd.conf

ファイルがどのパッケージに含まれるかを調べる。

「rpm -qf hoge」のように対象が見つからなければ「エラー: ファイル /home/vagrant/hoge: ファイルまたはディレクトリが見つかりません」と表示される。

yum

yum info [yumパッケージ名]

例:yum info php

PHPのパッケージについて調べる

yum install [yumパッケージ名]

例:yum install epel-release

rpmパッケージをインストール

yum list [all | パッケージ名]※ パッケージ名は省略可能

全ての利用可能なパッケージの一覧を表示する

yum list installed [パッケージ名]※ パッケージ名は省略可能。

システムにインストール済みの全てのパッケージを一覧表示する

yum list available [パッケージ名]※ パッケージ名は省略可能。

リポジトリ上の利用可能なパッケージを一覧表示する

yum grouplist

全てのパッケージグループを一覧表示する(いつくかのパッケージは目的ごとにグループ分けされています。)

cat /etc/redhat-release

Linux RedHatバージョン確認

用語

rpmコマンド

RedhatLinux社が開発したパッケージ管理のコマンドです。 Linuxのソフトウェアを簡単に管理することができます。 RPM形式は、一種の圧縮されたアーカイブで、パッケージされたファイルや参照するファイルのデータベースを保持しています。

RedHat

Linuxのディストリビューションを提供しているRedHat社とRedHat社が提供しているディストリビューションRHEL（Red Hat Enterprise Linux）を指すことが多い。RedHat社はLinuxの有償ディストリビューション提供企業としては最大手企業だ。

rpmとyumの違い

rpm（個々のパッケージを扱う）

yum（リポジトリから依存関係を含めて管理する）

【参考】【初心者にもわかる】rpmとyumの違いと使い分け一通り

epelリポジトリ

EPELとは、業務用のサーバなどで利用するエンタープライズ向けLiunx用のソフトウェアを集積したリポジトリの一つ。 CentOS等の公式パッケージに含まれないが有用なソフトウェアを提供している。
