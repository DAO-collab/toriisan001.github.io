CSS

cssは長くなりがちなのでコメントを書いておくと便利

        
/* common */
/* header */
/* main */
/* footer */
/* side */
/* animation */

バックグラウンド色を変更

        
background: #efefef;

bodyタグ全体のマージンを0にする

        
/* common */
body {
    margin: 0;
}

画像下の余白を消す

        
img {
    vertical-align: bottom;
}

タグの中のタグを指定

        
header .container {
}

幅を指定

        
width: 400px;

左右にマージンを設定して画面中央に配置

        
margin: 0 auto;

目安にするためアウトラインを設定

        
outline: 2px solid red;

横並びに表示

        
1.divの中にdivを入れる
<div class="container">
    <div>
        <h1>文字列１</h1>
    </div>
    <div>
        <h1>文字列２</h1>
    </div>
</div>

2.「display: flex;」を指定する
.container {
display: flex;
}

※ul要素にflexを付けることもできる

文字の太さを指定

        
font-weight:normal;

文字のサイズを変更

        
font-size: 24px;

リストのスタイルをなくす

        
header ul {
    list-style: none;
}

要素に続く要素に対して属性を指定

        
header ul li + li {
    margin-left: 8px;
}

親要素に対して中央ぞろえ

        
align-items: center;
text-align: center;
※どう違うのかはよくわからん

要素を丸くする

        
<img
src="../img/tottori.png"
alt="鳥居涼太のアイコン"
width="120"
height="120"
class="icon">
header .icon {
border-radius: 50%;
}
※画像、ボーダーラインなどに使用できる

余白

        
margin: 0 auto;
padding: 0 auto;
それから margin の場合、 margin の相殺が起きると面倒なので、どちらでもいいときは padding を選ぶ人も多い

boxの高さ（文字の行間を調節）

        
line-height: 1.8;
