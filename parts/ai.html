AIの種類

ChatGPT

OpenAIが2022年11月に公開したチャットボット。OpenAIのGPT-3ファミリーの言語モデルを基に構築されており、教師あり学習と強化学習の両方の手法で転移学習されている。2022年11月30日にプロトタイプとして公開され、幅広い分野の質問に詳細な回答を生成できることから注目を集めた。

基本無料で利用できる。月額20ドル（約2,500円）のサブスクリプションプラン｢ChatGPT Plus｣も発表されており、ピーク時でもアクセスしやすくなったり、応答時間が短縮されたり、新機能などが使えたりするとのことです。

ChatGPT使い方総まとめ

RPA(ロボティックプロセスオートメーション(Robotic Process Automation))

ロボティック・プロセス・オートメーションは、ソフトウェアロボット または仮想知的労働者と呼ばれる概念に基づく、事業プロセス自動化技術の一種である。デスクトップ作業のみに絞ったものをロボティック・デスクトップ・オートメーションと呼び、RPAと区別することもある。

Bing AI

BingのAIは、Bing検索に統合された会話型のAIサービスです。 BingのAIは、ユーザーの言語や興味に合わせて、情報やエンターテイメントを提供します。 BingのAIは、検索結果やウェブ上の情報をもとに、論理的で正確な回答を生成します。

Novel AI(ノベルエーアイ)

NovelAIは、オンラインクラウドに基づくSaaSモデルの有料サブスクリプションサービスで、AIによってシチュエーションと文章から画像の出力を行う。2021年6月15日にベータ版が公開され、2022年10月3日に画像生成機能が実装された。

ControlNet

AIでのイラスト生成時、キャラクターに自由なポーズを取らせることができる。

東北きりたんの声優は、日本の女性声優である矢作紗友里（やはぎ さゆり）さんです
    
チャットにスレッドがあって
スレッド間で情報共有していないらしい。
同じデータを参照している別のインスタンス？らしい。
東北きりたんの声優は新しいインスタンスになるとわからなくなってしまう。

コード生成

JavaScript例 
鳥居
うるう年を判定するJavaScriptの関数を出力して
以下はJavaScriptでうるう年を判定する関数です。

javascript
Copy code
function isLeapYear(year) {
  if (year % 4 !== 0) {
    return false;
  } else if (year % 100 !== 0) {
    return true;
  } else if (year % 400 !== 0) {
    return false;
  } else {
    return true;
  }
}
この関数は、引数として与えられた年がうるう年かどうかを判定し、うるう年であれば true を、そうでなければ false を返します。うるう年の定義については、以下のようになっています。

4で割り切れる年はうるう年
ただし、100で割り切れる年はうるう年ではない
ただし、400で割り切れる年はうるう年
例えば、以下のようにして関数を呼び出すことができます。

javascript
Copy code
console.log(isLeapYear(2000)); // true
console.log(isLeapYear(1900)); // false
console.log(isLeapYear(2024)); // true
console.log(isLeapYear(2021)); // false
この場合、最初の呼び出しでは2000年がうるう年であるため、 true が出力されます。2番目の呼び出しでは1900年がうるう年ではないため、 false が出力されます。3番目と4番目の呼び出しでは、それぞれうるう年であるかどうかが判定され、結果が出力されます。
