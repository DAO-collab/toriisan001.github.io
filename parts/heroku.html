HerokuCLI

ログイン

        
heroku loign

bash参照

        
heroku run bash -a ${アプリ名}

REDISインスタンスの情報を確認

        
heroku redis:info -a ${アプリ名}

環境情報を確認

        
heroku config -a ${アプリ名}

postgres

postgres接続


heroku pg:psql -app ${アプリ名}

DBから取得


\o output.csv // 記録開始
\a // エンコードだっけ
\f ',' // 区切り文字の設定
\i ${ファイル名} // ファイル読み込み。そのままsql書いてもいい
\o // ファイル出力

UTF-8に変換(変換しないとDB取得が出来ないことがあった)


chcp 65001

テーブル定義の確認


\d ${テーブル名}

スタックの確認


heroku stack --ap ${アプリ名}
