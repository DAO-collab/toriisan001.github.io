PyScriptを試してみました【参考】

pyScriptはWebAssemblyを活用していてPythonをブラウザ上で実行できるようにするオープンソースソフトウェアらしい

まだアルファ版でありプロダクションには使用しないよう勧めるらしい

使い方

py-script

パイソン埋め込み

            
        <py-script>print("Hello, world!")</py-script>

py-repl

ユーザーがパイソンを実行可能。コードを打ち込むと右端に緑の再生ボタンで実行可能らしい(Shift + Enterでも可)

py-env

環境変数らしい。使い方はよくわからない。

パイソンコマンド(簡単なもの)

文字列出力

            
        print("Hello, world!")

変数に設定。出力(文字列)

            
        name="shimada"
        print("hello %s" %name)

※エンターキーで改行が可能

コメント

ctl + / でコメントにできる

            
        # commentを記載

ユーザーからの入力を受け付ける

            
        print("Your name?", end="")
        name = input()
        print("hello %s" %name)

整数値の場合は%sではなく%iを使用する

printで改行させない

            
        print("Your name?", end="")
        name = input()

ユーザーから数値を受け取って計算した値を表示

            
        print("Your number?", end="")
        num = int(input())
        print(num + 3)

※input()で受け取った文字列は文字列なので数値化する必要がある。

if else文

            
        answer = 6

        print("Your guess? ", end="")
        guess = int(input())
        
        if answer == guess:
            print("Bingo!")
        else:
            print("Boo...")

※スペース4つが字下げとなる

else if文

            
        answer = 6

        print("Your guess? ", end="")
        guess = int(input())
        
        if answer == guess:
            print("Bingo!")
        elif answer > guess:
            print("Bigger!!")
        else:
            print("Smaller!!")

ランダムな数を読み込む

            
        import random
        answer = random.randint(1, 10)
        
        print("Your guess? ", end="")
        guess = int(input())
        
        if answer == guess:
            print("Bingo!")
        elif answer > guess:
            print("Bigger!!")
        else:
            print("Smaller!!")
        
        print("[Answer: %i]" %answer)

正解するまでループさせる

            
        import random
        answer = random.randint(1, 10)
        while True:
            print("Your guess? ", end="")
            guess = int(input())
            
            if answer == guess:
                print("Bingo!")
                break
            elif answer > guess:
                print("Bigger!")
            else:
                print("Smaller!")

回数を保持する

            
        import random

        answer = random.randint(1, 10)
        count = 0
        
        while True:
            print("Your guess? ", end="")
            guess = int(input())
            count += 1
            if answer == guess:
                print("Bingo! It took %i guesses" %count)
                break
            elif answer > guess:
                print("Bigger!")
            else:
                print("Smaller!")
